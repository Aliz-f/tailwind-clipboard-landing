# Clipboard Landing Page

A sleek, fully responsive product landing page for **Clipboard**, a fictional cross-platform clipboard management
application. This project was built to master semantic HTML5 layouts and modern utility-first styling workflows.

## 🚀 Features

- **Mobile-First Responsive Design:** Smooth layouts adapting fluidly from mobile displays up to ultra-wide desktop
  monitors.
- **Modern Typography & Components:** Uses the *Bai Jamjuree* typeface with optimized hover/active states for
  interactive elements.
- **Tailwind CSS v4 Architecture:** Built using the latest Tailwind CSS v4 engine utilizing standalone CLI setups and
  modern css-based configurations.

## 🛠️ Built With

- **HTML5:** Semantic architecture (`<section>`, `<footer>`, etc.)
- **Tailwind CSS v4:** Fast, utility-first styling workflow
- **Google Fonts:** Integration of custom design typography
